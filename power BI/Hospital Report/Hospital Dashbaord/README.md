Hospital Dashboard Summary
Overview
This dashboard provides a comprehensive overview of hospital performance, patient demographics, and operational efficiency. Built using power BI and Excel, the report highlights critical KPIs, and visuals insights to support data-driven decisions.

 Tools used
•  Power BI Desktop 
•  Excel 
•  Data modelling: date table, Status table, department table, Age band table

•  Dax Measures including: Total Patients, Admission, Rate, Average waiting Time, satisfaction score, and other aggregated metrics
•  Visualization: Cards, Bar charts, Donut chart, slicers

Key Features 
•  KPI summary panel displaying patient volume, admission rate, waiting time and satisfaction score
•  Patient distribution by race, gender, age band, department, and day of the week
•  Interactive slicers for annual comparison (2021 – 2023)
•  Clean visuals with a custom color theme for clarity and storytelling 

Key insights
•  Total Patients: 688
•  Admission Rate: 49.56%, indicating moderate admission volume 
•  Average waiting time: 39.67 minutes, highlighting a need for operational efficiency 
•  Average satisfaction score: 6.50, suggesting room for patient experience improvement
•  Gender, age band and departmental distributions show clear demand patterns  

Business recommendations 
•  Reduce waiting time by improving triage processes and allocating more staff during peak days 
•  Enhance patient satisfaction through improved communication and better waiting-area conditions
•  Review resource distribution across departments to balance workload

Next steps 
•  Use insights to realign inventory strategy across regions ensuring high-demand areas receive priority allocation
